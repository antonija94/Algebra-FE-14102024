Formati grafickih datoteka

Za prikaz grafike na webu koriste se sljedeci formati:

JPG
JPG is an image file with the extension
GIF
GIF is a raster file format designed for relatively basic images that appear mainly on the internet
PNG
PNG is short for Portable Network Graphic, a type of raster image file
SVG
SVG is an XML-based markup language for describing two-dimensional based vector graphics
